## Installation

Clone 

```sh
git clone https://github.com/RevoU-FSSE-2/week-11-enkod-id.git
```

Windows:

```sh
import database 
```

text editor:

```sh
open visual studio code drag source code to text editor & open terminal 
```

run:

```sh
nodemon app.js 
```
